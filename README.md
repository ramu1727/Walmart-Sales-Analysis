# Walmart-Sales-Analysis
## 🛒 Walmart Sales Data Analysis
## 📌 Project Overview

Analyzed Walmart sales data to uncover key business insights

Focused on revenue trends, customer behavior, and sales patterns

Used SQL to extract, transform, and analyze transactional data

## 🎯 Objectives

Identify top-performing product lines and revenue drivers

Understand customer purchasing behavior

Analyze sales trends across different time periods

Support data-driven decision making for business operations

## 🛠️ Tools & Technologies

SQL (MySQL) – Data analysis and querying

Excel / CSV – Data storage and preprocessing

## 📊 Key Analysis Performed

Sales analysis by product line, branch, and city

Customer segmentation based on type, gender, and payment method

Revenue and VAT contribution analysis

Time-based analysis (daily, weekly, hourly trends)

## 🔍 Key Insights

Identified top-performing product categories contributing highest revenue

Evening hours recorded the highest number of transactions

Certain cities generated higher VAT, indicating strong sales performance

Customer segmentation revealed patterns in payment preferences and spending behavior
